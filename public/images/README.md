# Images Directory

This directory contains all the images used in your landing page.

## Recommended Images

### SEO & Social Media

- `og-default.png` (1200x630px) - Default Open Graph image
- `og-home.png` (1200x630px) - Home page specific Open Graph image

### Content Images

- Add your hero background images
- Logo files
- Feature illustrations
- Team photos
- Product screenshots

## Image Optimization Tips

1. **Use WebP format** when possible for better compression
2. **Optimize images** before adding them (use tools like TinyPNG)
3. **Use appropriate sizes** - don't use 4K images for small thumbnails
4. **Add alt text** in your components for accessibility

## Tools for Image Optimization

- [TinyPNG](https://tinypng.com/) - PNG/JPEG compression
- [Squoosh](https://squoosh.app/) - Image optimization
- [ImageOptim](https://imageoptim.com/) - Mac app for image optimization
